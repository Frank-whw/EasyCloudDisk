# Frontend Web前端 - 启动说明

## 🚀 快速启动

### Windows用户

**方法1：使用Python（推荐）**
```bash
# 双击 start.bat 或在命令行执行：
start.bat
```

**方法2：手动启动**
```powershell
cd frontend
python -m http.server 3000
```

然后浏览器访问：**http://localhost:3000** 或 **http://0.0.0.0:3000**

### Linux/Mac用户

```bash
cd frontend
./start.sh
```

## ⚠️ 重要提示

1. **不要直接双击 index.html 打开！**
   - 会出现CORS跨域错误
   - 必须通过HTTP服务器访问

2. **确保端口3000未被占用**
   - 如果被占用，修改端口号：
   ```bash
   python -m http.server 8080  # 改用8080端口
   ```

3. **后端服务器地址**
   - 默认连接：http://54.95.61.230:8080
   - 可在 `js/config.js` 中修改

## 🔧 常见问题

### Q: 显示"无法连接到服务器"
A: 检查后端服务器 http://54.95.61.230:8080/health 是否可访问

### Q: 文件上传失败
A: 
1. 检查文件大小（最大500MB）
2. 检查网络连接
3. 查看浏览器控制台错误信息

### Q: 需要自己的账号
A: 点击"注册"创建新账号，邮箱格式任意（如：test123@example.com）
