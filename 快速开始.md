# EasyCloudDisk - 快速开始指南

## 🎯 选择适合你的客户端

### 📱 场景一：我想随时随地访问文件（推荐新手）

**使用：frontend/ (Web前端)**

✅ 优点：
- 无需安装，打开浏览器就能用
- 支持手机、平板访问
- 轻量级，启动快

❌ 限制：
- 需要手动上传下载
- 关闭浏览器就停止

**启动方式：**
```bash
cd frontend
python -m http.server 3000

# 然后浏览器访问：http://localhost:3000
```

👉 详细说明：[frontend/启动说明.md](frontend/启动说明.md)

---

### 💻 场景二：我需要自动同步，像Dropbox一样

**使用：client/ (桌面同步客户端)**

✅ 优点：
- 自动监控本地文件夹
- 双向同步，无需手动操作
- 离线可访问本地副本

❌ 限制：
- 需要Java环境
- 需要编译运行
- 占用资源较多

**启动方式：**
```bash
# 1. 先修改配置（重要！）
# 编辑 client/src/main/resources/application.yml
# 改成你的邮箱和密码

# 2. 编译并启动
cd client
mvn clean package -DskipTests
java -jar target/clouddisk-client-1.0.0.jar
```

👉 详细说明：[client/启动说明.md](client/启动说明.md)

---

## 🚀 推荐使用流程

### 第一次使用

1. **先用 Web版 (frontend) 注册账号**
   ```bash
   cd frontend
   python -m http.server 3000
   # 访问 http://localhost:3000 注册
   ```

2. **上传一些测试文件**
   - 熟悉界面
   - 测试上传下载

3. **（可选）再启动桌面客户端 (client)**
   - 修改配置文件中的账号密码
   - 启动后会自动下载云端文件

### 日常使用

**推荐组合：**
- 🏠 **家里/办公室**：用桌面客户端（自动同步）
- 📱 **手机/出门**：用Web版（随时访问）

## 🐛 遇到问题？

### 常见问题速查

#### 1. Web版：打开后显示空白
- ❌ 不要直接双击 index.html
- ✅ 必须用 `python -m http.server 3000` 启动

#### 2. 桌面版：编译失败
- 检查Java版本：`java -version`（需要17+）
- 检查Maven版本：`mvn -version`（需要3.6+）

#### 3. 无法连接服务器
- 检查后端是否在线：访问 http://54.95.61.230:8080/health
- 应该返回：`{"database":true,"storage":true,"status":"UP"}`

#### 4. 桌面版无法登录
- 检查配置文件中的邮箱密码是否正确
- 确保账号已在Web版注册

#### 5. 文件没有同步
- 确保文件在 `client/local/` 文件夹内
- 查看控制台日志
- 检查网络连接
